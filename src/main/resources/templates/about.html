Despre Proiectul Meu

Bine ați venit pe site-ul meu! Proiectul meu are ca scop gestionarea informațiilor legate de clienți, reparații și inventar. Iată câteva dintre funcționalitățile proiectului meu:

Interogări

Explorează datele din baza noastră de date utilizând cele 8 interogări disponibile. Descoperă informații utile și analizează datele.

Operații CRUD

Proiectul meu permite efectuarea operațiilor Create, Read, Update și Delete (CRUD) pe fiecare tabel din sistem:

  • Persoana: Adaugă, modifică sau șterge informații despre persoane.
  • Deviz: Gestionează reparații, adăugând, actualizând sau ștergând detalii.
  • Piesa: Administrează stocul de piese, efectuând operații CRUD pe elementele de inventar.
  • Piesa_Deviz: Conectează reparațiile cu piesele folosite și ajustează cantitățile și prețurile de vânzare.

Te invit să explorezi funcționalitățile și să descoperi cum proiectul meu poate facilita gestionarea informațiilor.

Despre Proiect

Proiectul meu a fost construit folosind o serie de tehnologii moderne și eficiente. Iată o scurtă descriere a principalelor tehnologii utilizate în dezvoltarea proiectului:

  1. Baza de date Oracle:
    • Oracle este o bază de date relațională puternică și scalabilă.
    • Am utilizat Oracle pentru a stoca și gestiona datele referitoare la clienți, reparații, piese și relațiile dintre acestea.
  2. Java cu Spring Boot:
    • Java este un limbaj de programare robust și versatil.
    • Spring Boot este un cadru Java care facilitează dezvoltarea de aplicații rapide și ușor de întreținut.
    • Am folosit Spring Boot pentru a dezvolta backend-ul aplicației și pentru a oferi API-uri pentru interacțiunea cu frontend-ul.
  3. Thymeleaf:
    • Thymeleaf este un motor de șablon pentru limbajul de programare Java.
    • Am folosit Thymeleaf pentru a genera pagini web dinamice și pentru a afișa date din Java în pagini.
  4. Bootstrap:
    • Bootstrap este un cadru front-end bazat pe HTML, CSS și JavaScript.
    • Am integrat Bootstrap pentru a crea o interfață de utilizator modernă și responsivă.
  5. Alte tehnologii:
    • Am utilizat Git pentru gestionarea codului sursă, Maven pentru gestionarea dependențelor și diverse plugin-uri și librării Java pentru a extinde funcționalitățile proiectului.

Prin utilizarea acestor tehnologii, am creat o aplicație robustă și modernă care îmbină eficient funcționalitățile backend cu o interfață de utilizator atrăgătoare.
